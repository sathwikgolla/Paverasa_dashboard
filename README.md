# 🚀 Paverasa ERP – Finance & KPI Dashboard A modern **Finance & KPI Dashboard** developed for **Paverasa Pvt. Ltd.** to centralize financial management and provide real-time business insights. --- ## 📌 Overview The Paverasa ERP Finance Dashboard helps management monitor: * 💰 Revenue * 💸 Expenses * 📈 Profit & Loss * 💵 Cash Flow * 📊 Financial KPIs * 📉 Business Analytics * 📑 Reports --- ## ✨ Features ### 📊 Dashboard * Total Revenue * Total Expenses * Net Profit * Profit Margin * Cash Balance * Outstanding Payments * Pending Bills * Revenue Target Progress * Recent Transactions * Financial Insights * Interactive Charts ### 💰 Revenue Management * Add Revenue * Edit Revenue * Delete Revenue * Search Revenue * Filter Revenue * Payment Status Tracking ### 💸 Expense Management * Add Expense * Edit Expense * Delete Expense * Search Expenses * Filter Expenses * Vendor & Category Management ### 📈 Profit & Loss * Automatic Profit Calculation * Profit Margin Calculation * Profit/Loss Indicators ### 🔐 Authentication * User Registration * Secure Login * Password Hashing * Role-Based Access Control --- ## 👥 User Roles ### Finance Manager * Dashboard Access * Manage Revenue * Manage Expenses * View Profit & Loss ### Employee * Dashboard Access (Read Only) > **Note:** Admin accounts are created manually by the system administrator and cannot be registered through the application. --- ## 🛠 Tech Stack ### Frontend * Next.js * React.js * Tailwind CSS ### Backend * Next.js API Routes ### Database * MySQL ### Authentication * JWT * bcrypt ### Charts * Recharts --- ## 📂 Project Structure ```text app/ │── api/ │── dashboard/ │── revenue/ │── expenses/ │── profit-loss/ │── auth/ │   ├── login/ │   └── register/ │── components/ │── lib/ │── hooks/ │── utils/ ``` --- ## ⚙️ Installation Clone the repository ```bash git clone <repository-url> ``` Navigate to the project ```bash cd paverasa-dashboard ``` Install dependencies ```bash npm install ``` Create a `.env.local` file ```env DB_HOST=localhost DB_PORT=3306 DB_NAME=paverasa_dashboard DB_USER=root DB_PASSWORD=your_password JWT_SECRET=your_secret_key ``` Start the development server ```bash npm run dev ``` Open: ``` http://localhost:3000 ``` --- ## 🗄 Database Tables * Users * Revenue * Expenses * Targets --- ## 📦 Modules ### Module 1 – Dashboard * KPI Cards * Charts * Recent Transactions * Financial Insights ### Module 2 – Revenue Management * Revenue CRUD * Search * Filter ### Module 3 – Expense Management * Expense CRUD * Vendor Management * Categories ### Module 4 – Profit & Loss ``` Net Profit = Total Revenue − Total Expenses ``` ``` Profit Margin = (Net Profit / Total Revenue) × 100 ``` --- ## 🔒 Security * Password Hashing (bcrypt) * JWT Authentication * Protected Routes * Input Validation * Role-Based Authorization --- ## 🚀 Future Enhancements * Cash Flow Module * KPI Engine * Reports * PDF Export * Excel Export * CSV Export * Notifications * Audit Logs --- ##
